Первая работа посвящена изучению составного типа данных "Множество", а также  типа данных "Перечисление".
Необходимо  решить две задачи (одна задача на работу с множествами, одна задача на работу с перечисляемым типом).
Программная реализация полученных задач должна осуществляться с использованием соответсвующего типа данных.

Для успешной защиты первой работы необходимо:
- изучить теорию по теме работы;
- написать программы, опираясь не требования, перечисленные выше, разработать тесты и распечатать вместе с условием задачи;
- после успешной демонстрации программ преподавателю, ответить на вопросы,   касающиеся кода (алгоритм, структуры данных и т.д.);
- выполнить письменное задание по теме работы БЕЗ ОШИБОК.
