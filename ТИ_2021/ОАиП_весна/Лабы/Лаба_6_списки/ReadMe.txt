Hабота посвящена изучению типа "Запись", работе с типизированными файлами, работе с динамическими списками.

Теория: тип "Запись", работа с файлами (типизированные, текстовые), указатель, динамические переменные, динамические списки.



Требования к программе:
- программа должна работать в режиме  формы, а не консоли;
- интуитивно понятный пользовательский интерфейс;
- основные функции: загружение данных из типизированного файла в динамический список и работа с динамическим списком (создание, добавление, удаление, редактирование, сортировка, поиск), сохранение данных в типизированный файл.



Для успешной защиты работы необходимо:
- изучить теорию по теме работы;
- написать программу, опираясь не требования, перечисленные выше;
- после успешной демонстрации программы преподавателю,  ответить на   вопросы,   касающиеся темы работы;
- выполнить письменное задание по теме работы БЕЗ ОШИБОК.
