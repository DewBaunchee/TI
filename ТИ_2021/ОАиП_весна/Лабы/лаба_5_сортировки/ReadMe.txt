Работа посвящена изучению эффективности различных видов сортировок.

Алгоритмы сортировок смотреть в книге "Алгоритмы и структуры данных" Н.Вирт.

Самостоятельно разобраться в работе быстрой сортировки (рекурсивная версия).Схема алгоритма быстрой сортировки в файле "qsort.docх". Пример работы сортировки для массива чисел в файле "быстрая массив.doc".

Варианты заданий в файле "Задания.doc".

Требования к программе:
- программа должна работать в режиме формы, а не консоли;
- интуитивно понятный пользовательский интерфейс;
- построение гистограмм сравнений и перестановок, используя свойство   Canvas;
Результат работы программы: таблица, гистограммы сравнений и перестановок.

Содержание отчета:
- титульный лист;
- условие задачи;
- описание сортировок с примерами;
- схемы алгоритмов сортировок;
- анализ сортировок;
- таблица с результатами работы сортировок (сравнения, перестановки) + рассчетные значения сравнений и перестановок;
- вывод об эффективности сортировок.


Для успешной защиты четвертой работы необходимо:
- изучить теорию по теме работы;
- написать программу, опираясь не требования, перечисленные выше;
- оформить отчет согласно требованиям, перечисленным выше;
- после успешной демонстрации программы преподавателю,  ответить на   вопросы,   касающиеся темы работы;
- выполнить письменное задание по теме работы БЕЗ ОШИБОК.
